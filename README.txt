# Royal Furniture Website — Setup Instructions

## Files in this package

```
royal-furnitures/
│
├── Home.html          ← Home page
├── PRODUCTS.HTML      ← Products page (shop + cart)
├── About Us.html      ← About page
├── contact.html       ← Contact page
├── royal.css          ← All styles (updated)
├── royal.js           ← All JavaScript (new — interactivity)
└── images/            ← Your existing images folder (copy from GitHub)
```

## How to use

1. Download all files from this package.
2. Copy your `images/` folder from your GitHub repo into the same directory.
3. Open `Home.html` in your browser — everything works locally.
4. To publish online, upload ALL files (including `images/`) to your GitHub repo,
   replacing the old files.
